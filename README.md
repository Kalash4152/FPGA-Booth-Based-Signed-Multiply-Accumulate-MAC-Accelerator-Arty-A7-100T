# FPGA-Booth-Based-Signed-Multiply-Accumulate-MAC-Accelerator-Arty-A7-100T
A multi-cycle signed Booth multiplier integrated into a pipelined 32-bit accumulator and deployed on the Arty A7-100T FPGA. Inputs are provided via switches, execution is triggered by push-buttons, and results are displayed on LEDs, demonstrating a hardware MAC datapath suitable for DSP and AI workloads.
